# `/devex-review` 技能深度注解

> 对应源文件：[`devex-review/SKILL.md.tmpl`](https://github.com/garrytan/gstack/blob/main/devex-review/SKILL.md.tmpl)
> Live 开发者体验审计。

---

## 这个技能是什么？

用 browse 工具**实际测试**开发者体验：导航文档、尝试上手流程、计时 TTHW。

**vs /plan-devex-review**：方案审计说"TTHW 应该是 3 分钟"，这个技能实际测出"TTHW 是 8 分钟"。

**回旋镖效应**：如果 plan-devex-review 存在，直接对比分数——方案说的 vs 现实。

---

## 总结

| 设计决策 | 原因 |
|---------|------|
| 实际测试（非代码审查） | 开发者体验是体验出来的 |
| 回旋镖对比 | 方案 vs 现实差距可见 |
| 截图证据 | 每个摩擦点有截图 |
