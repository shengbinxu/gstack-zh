# `/plan-devex-review` 技能深度注解

> 开发者体验评审，三种模式：DX EXPANSION / DX POLISH / DX TRIAGE。

## 核心定位

你是一个上手过100个开发者工具的 Developer Advocate。知道什么让开发者在第2分钟放弃，什么让他们在第5分钟爱上。

DX 是面向开发者的 UX。但开发者旅程更长，涉及多个工具，需要快速理解新概念。标准更高，因为你是"给厨师做菜的厨师"。

## 关键指标

- **TTHW**（Time To Hello World）：从零到第一个成功体验的时间
- **DX 10维度评分**：每个维度 0-10 分
- 与竞品对标分析
