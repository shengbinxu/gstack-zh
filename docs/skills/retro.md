## 这个技能是什么？

分析 commit 历史 + 工作模式 + 代码质量指标 → 持久化历史 → 趋势追踪。

**团队感知**：多人贡献时按人细分，每人有 praise + 成长建议。

**消费 /ship 指标**：覆盖率趋势、发布频率等来自 /ship Step 8.75。
